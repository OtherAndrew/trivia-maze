- Find the goal to win the game!
- Move with the direction buttons on the bottom left,
  or use the WASD keys.
- To pass through the question doors, you must answer a question.
    - If you answer correctly, the door will unlock.
    - If you answer incorrectly, the door will lock.
    - Press cancel to choose a different door.
- If a door leading to the goal is locked then the game is lost.
- Save your game by using the "Q.Save" or "Save" button.
- Load a game by using the "Resume" button on the main menu
  or by pressing "Q.Load" or "Load."
